What is PyroCMS?

PyroCMS is an open source, object oriented content management system built built using PHP5 and CodeIgniter.

PyroCMS is licensed under the Apache license, so you can legally use it for any kind of open source, commercial, or personal project.

What makes PyroCMS great?

PyroCMS is easy to use, looks great and uses some smart caching to keep everything running smoothly. It can be easily extended with Modules, Widgets, Plugins which are easy to make thanks to CodeIgniter and it can be customized with Themes, which are basic HTML with a few tags in place.
