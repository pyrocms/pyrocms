pyro:blog

The blog tag displays blog posts.

Tags

{pyro:blog:posts}


Display all blog posts or blog posts by category.

Parameters

Parameter Default Description
category All Categories Optional. Display posts from this category only.
limit 10 Optional. The maximum number of posts to display.
order asc Optional. Display posts by asc, desc, or random.

Example

{pyro:blog:posts limit="5" order="desc" category="pyrocms"}
	<h2>{title} -- {created_on}</h2>
	<a href="http://yoursite.com/blog/view/{slug}">Read More</a>
	<p>{intro}</p>
	<p>{body}</p>
{/pyro:blog:posts}
