Partials

When you start creating multiple Layouts you will soon start to realize that while they are not all the same they will share lots of stuff. Often the header will be different but there could be two columns instead of one, etc.

Partials are created as addons/themes/xxx/views/partialname.html and are loaded with:

{pyro:theme:partial name="partialname"}

You can put any syntax or HTML into partials that you would normally put in a layout, they will all be treated the same.

Example

The easiest example of a layout using a shared partial is this:

<!DOCTYPE html>
<html>
<head>
	{pyro:template:partials:header}
</head>
<body>
	<h1>{$template.title}</h1>
	{pyro:template:body}
</body>

</html>
