Helper Plugin

The helper plugin gives you access to two handy functions for language and time.

The helper plugin's slug is helper, so it can be used like so:

{pyro:helper:function}

Functions

{pyro:helper:lang}

Displays a language line in the language of the current language.

Parameters

Parameter Description
line Required. The slug for the language line you want to display.

Example:

{pyro:helper:lang line="error_message"}
Error!

{pyro:helper:date}

Displays a formatted date for the current time or a timestamp you specify.

Parameters

Parameter Description
format Required. Format for the output. This plugin uses PHP's date or strftime formatting syntax.
timestamp UNIX timestamp to use instead of the current date.

Example:

{pyro:helper:date format="Y"}
2011

{pyro:helper:gravatar}

Displays an avatar using gravatar API.

Parameters

Parameter Default Description
email None Required. The Email address used to generate the gravatar.
size 50 The size of the gravatar in pixels. A size of 50 would return a gravatar with a width and height of 50px.
rating g The rating of the gravatar. Possible values are g, pg, r or x.
url_only false true or false. Set this to TRUE if you want the plugin to only return the gravatar URL instead of the HTML.

Example:

{pyro:helper:gravatar email="user@example.com" size="20"}
<img src='http://www.gravatar.com/avatar/b58996c504c5638798eb6b511e6f49af?s=20&r=g' alt='Gravatar' class='gravatar' />
